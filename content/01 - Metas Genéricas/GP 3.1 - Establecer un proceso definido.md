# GP 3.1 — Establecer un proceso definido

**Meta Genérica:** [[GG 3 - Institucionalizar un proceso definido|GG 3]]

Establecer y mantener la descripción de un proceso definido.

---

## Subprácticas

- **Subpráctica 1:** Seleccionar del conjunto de procesos estándar de la organización aquellos procesos que cubran el área de proceso y que mejor satisfagan las necesidades del proyecto o función de la organización.
- **Subpráctica 2:** Establecer el proceso definido adaptando los procesos seleccionados de acuerdo con las guías de adaptación de la organización.
- **Subpráctica 3:** Asegurar que los objetivos del proceso de la organización se tratan de forma apropiada en el proceso definido.
- **Subpráctica 4:** Documentar el proceso definido y los registros de la adaptación.
- **Subpráctica 5:** Modificar la descripción del proceso definido según sea necesario.
