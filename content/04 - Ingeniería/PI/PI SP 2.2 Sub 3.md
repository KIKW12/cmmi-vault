## Descripción

Mantener un repositorio para los datos de interfaz accesible a los participantes del proyecto. Un repositorio común accesible para los datos de interfaz proporciona un mecanismo para asegurar que todo el mundo conoce dónde residen los datos actuales de interfaz y puede acceder a ellos para su uso.
