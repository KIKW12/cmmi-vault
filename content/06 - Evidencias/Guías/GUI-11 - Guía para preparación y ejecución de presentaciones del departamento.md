CONSIDERACIONES IMPORTANTESLa planeación de la presentación debe seguir el proceso de creatividad documentado por el departamento.La asignación de presentadores debe seguir el proceso ya definido por el departamento.No usar mochilas, bolsas, celulares o accesorios visibles durante la presentación.Se debe llegar con anticipación suficiente para montaje y revisión técnica (Al menos 45 minutos antes en el caso de los presentadores y 20 minutos integrantes que no presentarán).La información de la invitación debe coincidir con la logística real del evento.Si cambia el lugar, horario o dinámica del evento, la guía debe revisarse y ajustarse.LINEAMIENTOS DE IMAGENEl código de vestimenta para este tipo de eventos es semi formal.Para el código de vestimenta, se debe consultar el tablero de Pinterest de referencia visual.No se permiten jeans desgastados, que no sean de color sólido, o demasiado ajustadosNo se permiten escotes grandesNo se permiten gorras No se permiten transparenciasNo se permiten lentes de solCuidar que la ropa este en condiciones y sin demasiadas arrugasSí se permite el uso de linoEl calzado debe estar limpioNo se permite calzado deportivoNo se permiten sandaliasSí se permiten botas (no de uso rudo)Todos deben asistir peinados y con una imagen pulcraEn el caso de las mujeres, se permite llevar el cabello suelto siempre que esté bien arreglado y peinado.En el caso de los hombres, asistir rasurados o con barba/bigote bien recortados y arreglados.Cuidar la higiene personal y usar una presentación limpia y ordenada.Evitar prendas con estampados muy grandes, logos excesivos o mensajes.Procurar colores neutros o sobrios para mantener una imagen profesional.
