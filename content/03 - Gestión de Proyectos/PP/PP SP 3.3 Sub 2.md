# PP SP 3.3 — Subpráctica 2

**Área de Proceso:** [[PP]] · **Meta:** [[PP SG 3 - Obtener el compromiso con el plan|SG 3]] · **Práctica:** [[PP SP 3.3 - Obtener el compromiso con el plan|SP 3.3]]

## Descripción

Documentar todos los compromisos de la organización, tanto definitivos como provisionales, asegurando el nivel apropiado de firmantes. Los compromisos deberían documentarse para asegurar una comprensión mutua consistente y para el seguimiento y mantenimiento del proyecto. Los compromisos provisionales deberían acompañarse de una descripción de los riesgos asociados.
