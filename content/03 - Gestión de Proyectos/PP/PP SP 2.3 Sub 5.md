# PP SP 2.3 — Subpráctica 5

**Área de Proceso:** [[PP]] · **Meta:** [[PP SG 2 - Desarrollar un plan de proyecto|SG 2]] · **Práctica:** [[PP SP 2.3 - Planificar la gestión de los datos|SP 2.3]]

---

## Descripción

Decidir qué datos y planes del proyecto requieren control de versión u otros niveles de control de configuración y establecer mecanismos para asegurar que los datos del proyecto se controlan.

## Contexto del Departamento

> **Objetivo (Departamento):** El objetivo de esta práctica es identificar, recopilar, almacenar, y proteger los datos que se generan durante el proyecto.
>
> **Artefactos:** Plan de Gestión de datos. Requisitos de Privacidad. Requisitos de Seguridad. Proceso para recuperación, reproducción y distribución de datos.
