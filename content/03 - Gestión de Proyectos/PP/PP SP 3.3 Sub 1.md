# PP SP 3.3 — Subpráctica 1

**Área de Proceso:** [[PP]] · **Meta:** [[PP SG 3 - Obtener el compromiso con el plan|SG 3]] · **Práctica:** [[PP SP 3.3 - Obtener el compromiso con el plan|SP 3.3]]

## Descripción

Identificar el soporte necesario y negociar los compromisos con las partes interesadas relevantes. La WBS puede usarse como una lista de comprobación para asegurar que se obtienen los compromisos para todas las tareas. El plan para la interacción con las partes interesadas debería identificar todas las partes de las que se debería obtener el compromiso.
